VOD識別（配信サービス）

① 対象ライブラリを選ぶ（複数選択可）

読み込み中…
ここで選んだライブラリの中だけをスキャンします。何も選んでいないとスキャンは実行されません。

② TMDB APIキー

themoviedb.org のアカウント設定 → API から取得できる v3 の「APIキー」。 各作品の配信元(Netflix等)を日本地域(JP)で調べるのに使います。

③ 実行

設定を保存してから、選んだライブラリだけを走査します（作品数によっては数分かかります）。

ワンタップ接続

Jellyseerr URL(任意)

設定すると、ワンタップ接続(Connectページ)経由でこのURLもアプリに渡され、 アプリ内で同じJellyfinの資格情報を使ってJellyseerrにも自動的にログインします。 空欄のままなら何も渡されません。

ストアURL(任意・上書き)

アプリ未インストール時の「ストアでインストール」リンク先。接続したデバイスのOSを自動判定してiOS/Androidどちらかに振り分けます。 空欄のままなら既定のC-filmアプリの配布URLを使うので、通常は入力不要です。
